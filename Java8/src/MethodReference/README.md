# Method Reference

## Decription
- 하나의 추상 메소드로 이루어진 인터페이스

## Features
- Static Method Reference
 >- 파라미터와 리턴하는 변수가 같은 경우 사용
 >- 클래스::메소드명 형식

- Instance Method Reference
 >- 파라미터가 객체(클래스)이고, 해당 객체의 멤버를 리턴하거나 해당 객체의 메소드를 실행하는 경우 사용 
 >- 클래스::메소드명(멤버명) 형식

- Constructor Method Reference
 >- 새로운 객체를 생성하는 경우 사용
 >- 클래스명::new 로 형식

- Outer Instance Method Reference
 >- 객체의 메소드를 실행하는 경우 사용
 >- 객체::메소드명 형식
