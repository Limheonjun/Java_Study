# Lambda Expressions

## Decription
- 익명 함수를 지칭하는 용어이며 '->'를 사용해 표현

## Features
- 람다 표현식으로 구현이 가능한 인터페이스는 추상 메소드가 1개뿐인 인터페이스만 가능
- 단순한 람다 구문의 경우 람다 구분에 중괄호가 없을 수도 있음

## Functional Interface
- 추상 메소드가 1개뿐인 인터페이스
- @FunctionalInterface 애노테이션을 통해 해당 인터페이스가 함수형 인터페이스라는 것을 알려줌
- 추상 메소드가 1개가 아닐 경우 에러 발생
- 인터페이스의 용도를 알리기 위해 애노테이션을 붙임
