# String Methods

## Decription
- Java12에서 추가된 새로운 String Method

## Features
- indent : 파라미터로 받은 숫자만큼 들여쓰기
- transform : String클래스에 람다함수를 사용하여 변경할 수 있음
- describeConstable : JVMS 4.4 상수풀에 정의된 상수를 표시하는데 사용됨
- resolveConstantDesc(MethodHandles$Lookup) : java.lang.constant.ConstantDesc에서 수동으로 자주 호출하지 않는 저수준 JVM 상수 API를 지원하는 다른 방법의 상수 풀.
