# License Change

## Description
- Java11부터 라이센스 변동

## Features
- Oracle JDK와 OpenJDK가 동일해짐
- Java11부터 LTS버전으로 3년마다 출시, 5년동안 기술지원, 3년까지 지원기간 연장 가능. Oracle JDK가 3년마다 출시되니 Java의 실질적인 버전업을 담당하는 것은 Open JDK
- Open JDK는 기업들을 위한 기술 지원은 없고, 새로운 버전이 나오면 이전 버전에 대한 마이너 업데이트와 보안 업데이트는 중단
- Oracle JDK가 구독형 유료 모델로 전환
