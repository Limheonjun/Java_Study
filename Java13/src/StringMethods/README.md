# String Methods

## Description
-  Text Block과 연관된 새로운 String 메소드

## Features
- formatted : String클래스의 format 메소드와 유사. Text Block을 지원하기 위해 추가되었음
- stripIndent : Text Block의 모든 줄의 시작과 끝에서 우발적 인 공백 문자를 제거하는 데 사용. 이 방법은 텍스트 블록에서 사용되며 내용의 상대적 들여 쓰기를 유지
- translateEscapes : 문자열 리터럴로 변환된 탈출 시퀀스와 함께 이 문자열 값을 가진 문자열을 반환. (상위 버전에서 삭제될 예정)
